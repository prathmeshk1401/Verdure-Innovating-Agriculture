# üå± Verdure: A Smart Agricultural Assistant for Crop Monitoring and Care
Verdure is a full-stack, AI-powered platform designed to empower farmers and agricultural enthusiasts with intelligent tools for managing crop cycles, tracking harvests, and accessing actionable crop care insights. Built with a blend of frontend finesse, backend scalability, and NLP-driven smart logic, Verdure aims to make farming more data-aware, efficient, and sustainable.

# Features
- Crop Cycle Management: Add, monitor, and update crop stages with intuitive UI components.

- Harvest Tracking: Log harvests and visualize trends over time.

- Smart Insights (NLP): Rule-based logic to interpret crop health notes and suggest care actions.

- Geo-aware Recommendations (planned): Tailored advice based on location and crop type.

-  Modular Architecture: Scalable folder structure with dynamic imports and clean routing.

- Hackathon-Ready Demo Mode: Prioritized features for showcasing impact and usability
